If /health.cgi returns error:

                
Run in terminal:
python3 /opt/power-monitor/health.py /opt/power-monitor/config.json

If you see "No module named psutil":
apk add py3-psutil

If you see "No such file or directory config.json":
cp /opt/power-monitor/config.example.json /opt/power-monitor/config.json
vi /opt/power-monitor/config.json  # Edit with your settings


            

            

                
If /health.cgi returns 404:

                
Check file exists:
ls -la /var/www/html/health.cgi

Make executable:
chmod +x /var/www/html/health.cgi

Restart web server:
rc-service lighttpd restart
