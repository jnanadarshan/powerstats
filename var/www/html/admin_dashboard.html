build Configuration Setup Wizard

Run the setup wizard from the command line to reconfigure your system settings.

SSH into your device:

ssh root@your-device-ip

Run the setup wizard:

cd /opt/power-monitor
python3 setup_wizard.py

Or edit config directly:

vi /opt/power-monitor/config.json
warning
Important:

After making configuration changes, restart the services for changes to take effect:

systemctl restart power-monitor-scheduler
